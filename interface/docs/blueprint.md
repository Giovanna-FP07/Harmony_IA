# **App Name**: Harmony IA

## Core Features:

- Fatigue Score Calculation: Calculates a fatigue score based on biometric data and user input.
- 3D Body Mapping: Displays a 3D model highlighting areas of tension.
- AI-Powered Recommendations: Suggests personalized interventions (e.g., guided pauses, stretching) based on real-time data analysis.
- Real-Time Chat with AI Assistant (Aura): Enables users to interact with the AI for support and feedback using a tool to help identify context and extract action items.
- Customizable AI Monitoring: Allows users to adjust the sensitivity of facial detection and fatigue alert thresholds.
- Wellness Library: Offers resources such as breathing exercises, stretching videos, and relaxing music.
- WhatsApp Integration: Provides option to receive alerts and tips directly on WhatsApp for continuous care.

## Style Guidelines:

- Primary color: Pale purple (#CBC3E3) for a calming and soothing effect.
- Background color: Very light purple (#F5F0FF) for a clean and modern aesthetic.
- Accent color: Pale yellow (#F0E68C) to highlight interactive elements.
- Body and headline font: 'Inter', a sans-serif font for clear and accessible text.
- Use clean and simple icons to represent features and actions.
- Employ a card-based layout for a structured and organized presentation of information.
- Use subtle animations and transitions to provide feedback and guide the user.
